# Консольное приложение "T-Библиотека"

Программа для управления библиотекой книг. Работает в командной строке, хранит данные в JSON-файле. Её можно использовать в качестве личной пометки. Если вы начали читать книгу, то вы можете записать это в программу и делать личные пометки о ней, добавлять в избранное или изменять её статус.

## Возможности
- Просмотр всех книг с фильтрацией по жанру и статусу (прочитана/не прочитана).
- Сортировка по названию, автору или году.
- Добавление новой книги.
- Редактирование и удаление книг (доступно только после ввода секретного кода, который написан ниже).
- Поиск по названию, автору, жанру, году, статусу.
- Список избранных книг.
- Сохранение библиотеки в файл `library.json` и загрузка из него.
- Возможность делиться файлом библиотеки с друзьями и коллегами.

## Технологии
- Python 3.6+
- Используются только стандартные модули: `json`, `os`, `sys`.

## Установка и запуск
1. Скачайте Python 3.6+ из интернета.
2. Скачайте файлы проекта в ОДНУ ПАПКУ.
3. Откройте терминал в папке с файлом `secondtask.py`.
4. Запустите программу командой:
   ``` python secondtask.py ```
